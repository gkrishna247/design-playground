# Gradient Cards

A CSS gradient card component experiment showcasing modern gradient backgrounds with interactive hover effects.

## Description

This design explores the use of CSS gradients to create visually striking card components. The cards feature smooth color transitions, subtle shadows, and engaging hover animations that bring depth and interactivity to the UI.

## Techniques

- **CSS Gradients**: Linear and radial gradients for background effects
- **Hover Transitions**: Smooth scale and shadow transformations
- **Color Theory**: Harmonious color palette selection
- **Box Shadows**: Layered shadows for depth perception
