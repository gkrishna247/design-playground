# Gradient Cards

Beautiful gradient hover cards with smooth color transitions and subtle shadow effects.

## Features

- Pure CSS gradients (no images)
- Smooth hover lift animation
- Box shadow depth on hover
- Responsive grid layout
- Back navigation to gallery

## Technologies

- HTML
- CSS (Gradients, Transitions, Grid)

## Techniques Used

1. **CSS Gradients**: Using `linear-gradient()` with 135Â° angle for diagonal flow
2. **Transform on Hover**: `translateY(-8px)` for the lift effect
3. **Box Shadow Animation**: Dynamic shadow that grows on hover
4. **CSS Grid**: `auto-fit` with `minmax()` for responsive columns
