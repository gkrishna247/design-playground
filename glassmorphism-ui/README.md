# Glassmorphism UI

Frosted glass effect components with backdrop blur and translucent layers, inspired by modern OS design.

## Features

- Backdrop blur (frosted glass effect)
- Animated floating color blobs
- Gradient text with clip
- Glass button with hover effects
- Back navigation to gallery

## Technologies

- HTML
- CSS (Backdrop-filter, Animations)

## Techniques Used

1. **Backdrop Filter**: `backdrop-filter: blur(20px)` for the frosted glass
2. **RGBA Backgrounds**: Semi-transparent white for the glass surface
3. **Animated Blobs**: CSS keyframes for floating background elements
4. **Gradient Text**: Using `background-clip: text` for gradient typography

## Browser Support

Requires browsers that support `backdrop-filter`:
- Chrome 76+
- Safari 9+
- Firefox 103+
- Edge 79+
